jQuery Reference

.hide() and .show()

Clicking the "Hide" button will hide this text. Clicking the "Show" button will show it.

.toggle()

Clicking the "Toggle" button will hide this text if shown and show this text if hidden.

.slideUp() and .slideDown()

Clicking the "Slide Up" button will slide this text up. Clicking the "Slide Down" button will slide it down.

.slideToggle()

Clicking the "Slide Toggle" button will slide this text up or down, depending on whether or not it is hidden.

.fadeOut() and .fadeIn()

Clicking the "Fade Out" button will fade this text out. Clicking the "Fade In" button will fade this text in.

.addClass() and .removeClass()

Clicking "Add Class" will add a CSS class to this text. Clicking "Remove Class" will remove it.

.before() and .after()

Clicking "Before" will add a paragraph before this text. Clicking "After" will add a paragraph after this text.

.append()

Click "Append" to add text to this paragraph.

.html()

Click "HTML" to change the HTML content of this paragraph.

.attr()

Click the "Attr" button to change the href attribute of this link: Search Engine.

.val()

Click the "Val" button to change the text inside the textbox.

.text()

Click the "Text" button to change this paragraph.

.data()

Click the "Data" button to get an alert with a hidden data value.
